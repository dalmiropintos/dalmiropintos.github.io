Objetivos

  • Ampliar mis conocimientos en el desarrollo web y aprender nuevas tecnologías relacionadas.
  • Crear un portafolio sólido de proyectos personales que demuestren mis habilidades como desarrollador web.
  • Contribuir a proyectos de código abierto para adquirir experiencia y aprender de otros desarrolladores.
  • Trabajar en proyectos colaborativos en un entorno de equipo y mejorar mis habilidades de comunicación.
  • Mejorar la accesibilidad web y crear sitios web que sean accesibles para todos los usuarios, incluyendo aquellos con discapacidades.

Habilidades

  • HTML: Experiencia en la creación de páginas web semánticas, uso de etiquetas y atributos HTML, y estructura de documentos HTML5.
  • CSS: Conocimientos en la creación de estilos y diseños de páginas web, uso de selectores, propiedades y valores de CSS3, y diseño responsivo.
  • JavaScript: Conocimientos en la creación de interacciones dinámicas en sitios web, uso de eventos, funciones, objetos y métodos de JavaScript.
  • Diseño gráfico: Conocimientos en el diseño de elementos gráficos para páginas web utilizando Adobe Photoshop e Illustrator.
  • Adobe Photoshop: Experiencia en la edición de imágenes, creación de efectos y optimización de imágenes para la web.
  • Adobe Illustrator: Conocimientos en la creación de gráficos vectoriales y elementos de diseño para la web.
  • Herramientas de desarrollo: Conocimientos en el uso de herramientas de desarrollo como Visual Studio Code, Git y GitHub.
